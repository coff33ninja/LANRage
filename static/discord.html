1. Webhook Setup

Get notifications when players join, leave, or start games.

How to create a Discord webhook:

  1. Open your Discord server
  2. Go to Server Settings ‚Üí Integrations
  3. Click "Create Webhook" or "View Webhooks"
  4. Click "New Webhook"
  5. Name it LANrage
  6. Select the channel for notifications
  7. Copy the Webhook URL
  8. Paste it below

2. Voice Chat Invite (Optional)

Share a Discord voice channel link with your party.

How to create a Discord invite:

  1. Open your Discord server
  2. Right-click on a voice channel
  3. Click "Invite People"
  4. Click "Edit Invite Link"
  5. Set "Expire After" to Never
  6. Set "Max Uses" to No Limit
  7. Copy the invite link
  8. Paste it below

3. Rich Presence

Show your LANrage status in Discord (requires pypresence package).

To enable Rich Presence:

  1. Install pypresence: uv pip install pypresence
  2. Restart LANrage
  3. Your Discord status will update automatically
