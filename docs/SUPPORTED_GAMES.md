# Supported Games

Inventory and caveats for game profile support.

## What This List Means

- profiles exist for listed titles/categories
- support quality may vary by environment and profile source

## Reliability Tiers

- validated profiles: tested in LANrage workflows
- community/custom profiles: useful starting point, verify locally

## Before Relying on a Profile

- confirm process name and port assumptions
- test host/join workflow with real peers
- verify broadcast/multicast behavior where required

## If Detection Fails

1. check active process/port metadata
2. adjust or create custom profile
3. re-test with status and metrics views

## Maintenance Notes

- keep profile metadata versioned and documented
- remove stale/incorrect profiles to avoid false matches
