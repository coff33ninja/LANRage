# Roadmap

Planned direction for LANrage evolution.

## Near-Term

- improve operational robustness and docs parity
- expand relay/control-plane deployment guidance
- strengthen diagnostics and alerting patterns

## Mid-Term

- deeper remote control-plane policy options
- continued performance and quality automation
- broader profile and UX polish

## Longer-Term

- expanded platform and deployment flexibility
- richer ecosystem integrations
- advanced session coordination features

## Planning Principles

- prioritize reliability and debuggability over feature volume
- ship increments with measurable operational impact
- keep docs and implementation aligned per release
