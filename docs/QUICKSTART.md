# Quick Start

Fastest path from clone to first party.

## 1. Install

1. Clone repository.
2. Create/activate virtual environment.
3. Install dependencies.
4. Run project setup if needed.

## 2. Launch

Run LANrage:

```bash
python lanrage.py
```

Open UI:
- `http://127.0.0.1:8666`

## 3. Host a Party

1. Open main UI.
2. Create party with a name.
3. Share party ID with friends.

## 4. Join a Party

1. Enter shared party ID.
2. Enter display name.
3. Join and wait for peer status to reach connected/degraded.

## 5. Verify Connectivity

Use status views to check:
- party membership
- peer latency
- connection strategy (direct/relay)
